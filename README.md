## Descrição do Projeto
O objetivo central deste projeto é a aplicação de técnicas de **Deep Learning** para resolver o problema da "sobrecarga de escolha" em marketplaces de grande porte. Utilizando o dataset público da Olist, implementamos um sistema de recomendação baseado em camadas de **Embedding**, capaz de prever a afinidade de um usuário com determinados produtos. A motivação reside em otimizar a jornada de compra, tornando-a mais intuitiva e personalizada, o que reflete diretamente no aumento da taxa de conversão e na satisfação do consumidor.

* **Área/Tema:** Recomendação de Produtos / E-commerce.
* **Opção:** Opção ML/DL/VC/PLN (Deep Learning).
* **Dataset:** Brazilian E-Commerce Public Dataset by Olist.
